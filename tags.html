Important HTML Tags

This page is designed as a quick reference to some of themost common tags covered in Learn Enough HTML to be Dangerous. By doing these, we will learn how to make HTML tables via table and related tags.

The tables below do not include all HTML tages but do list many of the popular ones.

Block Elements

Tag Name Purpose
h1–h6 headings incldue a heading (levels 1–6)
p paragraph include a paragraph or text
table table include a table
tr table row incldue a table row
th table header make a table header
td table data include a table data cell
div division define a block-level section in the document
header header label the page header
ol ordered list list elements in numerical order
ul unordered list list elements whose order doesnt matter, by default, bullet
li list item include a list item (for ol and ul)
blockquote block quotation formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasise make emphasised text <em>technical sophistication</em> technical sophistication
strong srong make strong text <strong>technical spohistication<strong> technical sophistication
a link add a link <a href="link">text</a> text
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code include markup or source code in monospace font <code>strong</code> strong
span span define inline section in the document <span>Call me Ishmael.</span> Call me Ishmael.

Document Tags

Tag Name Purpose
html html To begin and end a HTML skeleton
head header container defines metadata, that is data to tell the browser where to find other files used to display the page's content without having that information show up on actual page
body body part of HTML which gets displayed to the browser
title title cntents of page title
meta meta a void element to define for example, the character set
