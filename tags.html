Important HTML Tags

This page is designed as a quick reference to some of themost common tags covered in Learn Enough HTML to be Dangerous. By doing these, we will learn how to make HTML tables via table and related tags.

The tables below do not include all HTML tages but do list many of the popular ones.

Block Elements

Tag Name Purpose
h1–h6 headings incldue a heading (levels 1–6)
p paragraph include a paragraph or text
table table include a table
tr table row incldue a table row
th table header make a table header
td table data include a table data cell
